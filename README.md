## 🛠️ How to Run

```bash
g++ -o solution XX_problem_name.cpp
./solution
```

Or compile with optimizations (like competitive programming judges):

```bash
g++ -O2 -std=c++17 -o solution XX_problem_name.cpp
./solution
```
